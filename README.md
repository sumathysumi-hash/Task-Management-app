# Task Management App

A simple and intuitive web application designed to help users organize, track, and manage their daily tasks efficiently.

## 🚀 Features
* **Create Tasks:** Easily add new tasks with titles and descriptions.
* **Track Progress:** Mark tasks as pending, in-progress, or completed.
* **Responsive Design:** Works smoothly on both mobile devices and desktops.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
